## 主要实现

- 将 22 类一级意图及全部三级路径抽取为机器可校验 taxonomy。
- 多意图 Prompt 强制选择合法路径，并自动推导一级、二级父路径。
- 非法路径在对应一级意图子树内定向修复。
- 增加典型易混淆意图的边界规则。
- 意图识别固定 `temperature=0`。
- 使用 `other.unclear` 和显式澄清字段识别意图不明，将商品、订单参数缺失交给槽位模块。
- 默认澄清阈值由 0.70 调整为 0.50。
- 项目自动化测试 45 项全部通过。

## 重要说明

本次使用的是人工定义模板生成的冻结合成基准，能够验证优化方向和回归稳定性，但不能替代脱敏真实客服日志的双人标注盲测。上线前建议补充 300–500 条真实样本独立验收。
